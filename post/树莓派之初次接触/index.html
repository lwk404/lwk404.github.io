一、烧录系统

1、树莓派官网下载系统：https://www.raspberrypi.org/downloads/raspbian/

2、烧录软件：
etcher：https://github.com/balena-io/etcher/releases
Win32 Disk Imager：https://win32diskimager.download/

二、ssh与WiFi

1、SSH
插入SD卡，在boot目录下新建ssh文件

2、WiFi
插入SD卡，在boot目录下新建wpa_supplicant.conf，写入以下内容：

country=CN
ctrl_interface=DIR=/var/run/wpa_supplicant GROUP=netdev
update_config=1
 
network={
ssid="WiFi的名字"
psk="WiFi密码"
key_mgmt=WPA-PSK
priority=1 #优先级，数字越大优先级越高
}

三、连接树莓派

确保树莓派与你的设备处在同一局域网下，使用局域网扫描工具获得树莓派的IP地址，Windows可以在cmd输入arp -a获取树莓派的IP地址。

ssh pi@ip地址
密码为raspberry

四、vnc远程连接树莓派

1、ssh连接系统后，输入：sudo raspi-config ，回车。在弹出的界面中，选择：“Interfacing Options”，回车。然后选择 “VNC” 并开启(enable)它。

2、运行vncserver，会获得一个IP地址。

3、安装vnc客户端VNC Viewer，输入IP地址，输入用户名，密码。

五、使用Windows的远程桌面功能替代VNC

ssh连接系统后，输入：

sudo apt-get purge tightvnc xrdp
sudo apt-get install tightvncserver xrdp

启动 xrdp 服务，sudo /etc/init.d/xrdp restart，此时会启用相应端口，配置方面默认即可。

通过命令：netstat -tnl
查看 3350 3389 5910 这三个端口是否处于LISTEN。

打开Windows远程桌面，输入用户名，密码即可。
