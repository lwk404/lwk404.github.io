解决Linux屏幕撕裂的问题
@ 无聊真人 · Wednesday, Dec 11, 2019 · 1 minute read · Update at Dec 11, 2019

什么是屏幕撕裂

输入视频与显示器的刷新率不同步时，这与视频或图形渲染有关。这就是所谓的屏幕撕裂，在LCD显示器上，水平屏幕撕裂非常突出。 屏幕撕裂主要发生在播放视频以及拖动/移动或最大化/最小化窗口时。

造成原因

该桌面使用的窗口管理器没有采用显卡的加速功能，而是简单地采用了CPU直接渲染。

解决方案

在/etc/X11/xorg.conf.d/下创建20-intel.conf：

Section "Device"
    Identifier "Intel Graphics"
    Driver "intel"
    Option "TearFree" "true"
EndSection

这样问题就解决了。
